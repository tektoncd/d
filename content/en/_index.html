{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} Tekton is a powerful and flexible open-source framework for creating CI/CD systems, allowing developers to build, test, and deploy across cloud providers and on-premise systems. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fas fa-globe-asia" title="Standardization" %}} Tekton standardizes CI/CD tooling and processes accorss vendors, languages, and deployment environments. It works well with Jenkins, Jenkins X, Skaffold, Knative, and many other popular CI/CD tools. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-medal" title="Built-in best practices" %}} Tekton let you create cloud-native CI/CD systems quickly, leveraging advanced deployment patterns such as rolling, blue/green, canary deployment and/or GitOps workflow. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-feather" title="Maximum flexibility" %}} Tekton abstracts underlying implmententation so that you can choose the build, test, and deploy workflow based on your team's requirements. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section >}}

Tekton is a Google-developed open source project.
